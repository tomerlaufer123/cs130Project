A factory to convert Json objects to a list of trends.

Recursively calls the Trend constructor and constructs a List of all the trends returned by the API call.

@params 'json' is a List that contains the entire json object in its first field. @return Initialized Album object

Implementation

factory Album.fromJson(List<dynamic> json){
  if(json[0]['trends'] != null){
    List<Trend> trendList = new List<Trend>();
    for(var js in json[0]['trends']){
      trendList.add(Trend.fromJson(js));
    }
    return Album(
      trends: trendList,
    );
  }
  List<String> syns = new List<String>();
  for(var syn in json[0]['meta']['syns'][0]){
    syns.add(syn);
  }
  return Album(
    synonyms : syns,
  );
}
